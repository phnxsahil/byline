---
title: "The Agent Pipeline"
description: "How Byline's 5-agent LangGraph architecture works."
---

Byline doesn't just pass your text to an LLM and ask it to "write a tweet." Standard LLMs output generic, promotional slop when given this instruction.

Instead, Byline orchestrates a multi-agent state graph built on **LangGraph**. This architecture ensures specialized agents handle distinct stages of the publishing workflow.

## 1. Ingestion & Memory

When a milestone is captured (via the web dashboard, CLI, voice note, or GitHub webhook), it first hits the Memory layer. 

Byline uses **pgvector** to perform a semantic search against your past dispatches and voice profile. This grounds the upcoming generation in your specific context and style.

## 2. The Strategist Agent

The Strategist is the first node in the LangGraph pipeline. It evaluates the raw milestone and decides:

1. **Is this post-worthy?** (Scores 1-10. Only proceeds if >= 6).
2. **What is the best angle?** (e.g., `lesson_learned`, `build_log`, `opinion`, `technical_deep_dive`).
3. **Which platforms are appropriate?** (It might select LinkedIn and X, but skip Threads if the topic is too dense).

The Strategist outputs a JSON routing plan that dictates which Writer agents should fire.

## 3. The Writer Agents

Byline runs four concurrent platform specialists:
- **LinkedIn Writer**: Optimizes for professional storytelling, short paragraphs, and hook-led openings.
- **X (Twitter) Writer**: Focuses on punchy, opinionated hooks and thread structures. Enforces character limits.
- **Reddit Writer**: Formats posts as educational deep-dives. Strips out self-promotion and frames the content as a "lesson learned."
- **Threads Writer**: Optimizes for highly casual, raw, unpolished energy.

These agents run in parallel, applying your custom Voice Profile constraints to their specific platform formatting.

## 4. The Critic Agent

Every draft generated by the Writers must pass through the Critic. 

The Critic scores each draft out of 10 on:
- **Voice Match**: Does this actually sound like you?
- **Anti-Slop**: Flags and removes words like "delve", "excited to announce", or "game-changer".
- **Platform Fit**: Evaluates if the post violates platform norms (e.g., promotional links on Reddit).

If a draft fails the Critic's checks (especially on Reddit for self-promotion risk), the draft is flagged or outright rejected for the user to rewrite.

You can view the Critic's full breakdown and notes inside the **Desk** panel before approving any post.
